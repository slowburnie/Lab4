Лабораторна робота No 3

«Створення вбудованої довідкової системи»

Розробити довідкову систему для програми з лабораторної роботи No 2, довідкова система
складається з двох частин:
1. Підказки, що випливають (hint / tooltip):
- у візуальних компонентах – короткі;
- у спеціальному компоненті (StatusBar, Panel) – більш повна підказка.
2. Довідкова служба (help):
- зміст (викликається з пункту меню «Зміст»);
- картинки (екрани) форм з коментарями;
- переходи за гіперпосиланнями (із затиранням вмісту, без затирання – у додатковому
спливаючому вікні).
